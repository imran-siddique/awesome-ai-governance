# Contributing to Awesome AI Governance

Thank you for your interest in contributing!

## Guidelines

### What belongs here?

This list focuses on **AI agent governance, safety, and compliance**:

- ✅ Tools that enforce policies on AI agents
- ✅ Frameworks for agent safety and guardrails
- ✅ Identity and trust verification for agents
- ✅ Compliance automation (GDPR, HIPAA, SOC2)
- ✅ Multi-agent coordination with governance
- ✅ Research papers on AI safety

### What doesn't belong?

- ❌ General AI/ML tools without governance focus
- ❌ LLM wrappers without safety features
- ❌ Chatbots without compliance features
- ❌ Promotional content without real tools

### How to contribute

1. Fork this repository
2. Add your tool in the appropriate category
3. Use the format: `- [Name](url) - Brief description.`
4. Submit a pull request

### Quality standards

- Tool must be actively maintained
- Should have documentation
- Open source preferred (but not required)
- Must genuinely help with AI governance

## Questions?

Open an issue or reach out!
